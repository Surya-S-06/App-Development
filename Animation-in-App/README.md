# Animation_Video

An Android application featuring frame-by-frame drawable animations with synchronized audio playback. Built with Java and Material Design, it demonstrates animation controls triggered by user interaction combined with MediaPlayer audio integration.

## Features

- **Frame-by-frame Animation**: Uses AnimationDrawable to create smooth sequential animations
- **Audio Synchronization**: Integrated MediaPlayer for synchronized audio playback
- **User Interactive Controls**: Button-triggered animation and audio playback
- **Material Design UI**: Modern Android Material Design components
- **Edge-to-Edge Display**: Full screen support with system insets handling

## Technical Stack

- **Language**: Java
- **Platform**: Android (API 24+)
- **Build System**: Gradle
- **UI Framework**: AndroidX with Material Design 3
- **Android Version**: Target SDK 36, Min SDK 24

## How It Works

The app demonstrates:
1. Creating an AnimationDrawable programmatically
2. Adding multiple animation frames with timing
3. Playing animations in a loop or single shot
4. Running MediaPlayer on a separate thread for audio playback
5. Synchronizing visual and audio elements

## Installation

Clone the repository and open it in Android Studio:
```bash
git clone https://github.com/Surya-S-06/Animation-in-App.git
cd Animation_Video
```

Build and run on an Android device or emulator (API 24+).
